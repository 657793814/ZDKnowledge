### Phase B — 课本之外的拓展

每个知识点的 contentJson 增加以下板块：

| 板块 | 内容 | 目的 |
|------|------|------|
| **🧩 思维拓展** | 变式、推广、逆向思考 | 培养数学思维 |
| **🔗 跨域链接** | 融合其他章节/学科的知识 | 建立知识网络 |
| **📐 推导证明** | 定理的多种证明方法 | 知其所以然 |
| **🌍 数学史话** | 定理发现的故事、数学家 | 增加趣味 |
| **🤖 思考题** | 开放式问题，无标准答案 | 激发思考 |

### Phase C — 试卷洞察（新功能）

```
用户上传试卷图像 → OCR识别题目文本 → AI分析每个题目的考点
→ 生成试卷洞察报告（考点分布热图、难度分析、知识漏洞）
→ 根据洞察自动组卷（强化薄弱知识点）
→ 用户反复练习，追踪进步
```
