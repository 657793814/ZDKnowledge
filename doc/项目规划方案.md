## 一、项目概述

### 1.1 愿景

打造一个 **图文并茂、环环相扣** 的初中高中数学学习平台。让每个知识点不再是孤立的小点，而是一张可以追溯、可以展开、可以互动的知识网。

### 1.2 核心理念

> **"一个数乘以 1 表示正向，乘以 -1 表示反向，乘以虚数 i 表示旋转 90 度。"**

用这样的直观表达方式，把抽象的数学概念变成可感知的图形和交互。每个知识点都回答三个问题：
1. **这是什么？**（定义 + 图示）
2. **从哪来？**（前置知识，层层追溯）
3. **到哪去？**（后置知识，学了能做什么）

### 1.3 核心特性

| 特性 | 说明 |
|------|------|
| 🔗 **知识连续性** | 每个知识点都知道前驱后继，形成知识图谱 |
| 🎨 **图文并茂** | SVG 动态图、动画、交互式 Canvas 演示 |
| 📊 **分层递进** | 按「数→式→方程→函数→几何→统计→微积分初步」层层递进 |
| 🧪 **训练+考试** | 知识点内练习 + 单元测验 + 综合模拟考 |
| 🌐 **多端覆盖** | Web（主） + 移动 App（辅助） |
| 🤖 **AI 辅助** | 知识点问答、错题解析、学习路径推荐 |

### 2.2 七大领域详细知识点链

#### 领域一：数（数的认识与运算）

```
自然数 ──→ 整数 ──→ 有理数 ──→ 实数 ──→ 复数
  │          │          │          │        │
  │          │          │          │     虚数旋转
  │          │          │     无理数(π,√2)  │
  │          │    分数/小数   │         a+bi
  │     负数(×-1=反向)  无限不循环    │
  │          │              │        i²=-1
  │          │              │        i=旋转90°
  │          │              │
  │  质数/合数/因数/倍数/公约数/公倍数
```

**关键可视化创意：**

- **数轴动态演示：** 1→-1 是绕原点旋转 180°，乘以 i 是旋转 90° → 用 Canvas 动画展示
- **复数平面：** 复平面上的向量旋转，欧拉公式 e^(iθ) = cosθ + i sinθ 的可视化

#### 领域二：式（代数式与运算）

```
整式 ──→ 分式 ──→ 根式 ──→ 指数式 ──→ 对数式
 │        │        │        │          │
单项式    分母≠0    √a      a^n       log_a(b)
多项式    分式运算  有理化   指数律    对数律
因式分解            n次根号   科学记数法 换底公式
```

#### 领域三：方程与不等式

```
一元一次方程 ──→ 一元二次方程 ──→ 分式方程 ──→ 方程组
     │              │               │            │
     │              │               │         二元一次
     │         判别式Δ=b²-4ac       │         三元一次
     │         求根公式             │
     │         韦达定理(根与系数)     │
     │              │               │
     └──────┬───────┴───────┬───────┘
            │               │
      一元一次不等式    一元二次不等式
      不等式组          含参不等式
```

#### 领域四：函数（核心领域）

```
变量与常量 ──→ 函数概念 ──→ 一次函数 ──→ 二次函数 ──→ 幂/指/对
  │                    │     y=kx+b      y=ax²+bx+c     │
  │             定义域/值域    │            │       指数爆炸增长
  │             解析法/图像    │       顶点/对称轴  对数缓慢增长
  │             列表法         │    │                      │
  │                           │    │                      │
  └── 在坐标系中，                             └── 导数基础
     ×1 是正向走                                  (切线斜率)
     ×(-1) 是回头走
     f(x+a) 是平移
     f(-x) 是镜像
```

**关键可视化创意：**
- **函数变换演示：** 拖动滑块改变参数，实时看图像变化（a 控制开口，b 控制对称轴）
- **平移/伸缩/对称 动画对比：** y=f(x) vs y=f(x+2) vs y=2f(x) vs y=f(-x)

#### 领域五：几何（平面 + 空间）

```
【平面几何】
点 ──→ 线 ──→ 角 ──→ 三角形 ──→ 四边形 ──→ 圆
                  │        │           │        │
              内角和180°  勾股定理   平行四边形  圆周角
              全等/相似   海伦公式   梯形/矩形   圆心角
              中线/高/角平分线        面积割补   弧长/扇形

【立体几何】← 从二维到三维的跃迁
点线面位置 ──→ 空间角/距离 ──→ 柱/锥/台/球 ──→ 向量法
    │              │              │              │
平行/垂直     线线角/线面角    体积/表面积   坐标法解立体
异面直线      二面角            祖暅原理       法向量

【解析几何】
坐标系 ──→ 直线方程 ──→ 圆方程 ──→ 圆锥曲线
  │           │           │          │
数形结合    点斜式/截距式  标准方程   椭圆/双曲线/抛物线
两点间距离  点到直线距离   一般方程   焦半径/离心率
```

**关键可视化创意：**
- **勾股定理无字证明：** a² + b² = c² 的面积拼图动画
- **立体几何展开图：** 正方体/圆锥/圆柱的 3D 展开演示（Three.js）
- **圆锥曲线统一性：** 平面截圆锥 → 得到椭圆/抛物线/双曲线 的动态演示

#### 领域六：排列组合与概率统计

```
分类计数(加法) ──→ 排列 ──→ 组合 ──→ 二项式定理
  │              P(n,k)    C(n,k)    (a+b)^n
分步计数(乘法)   全排列    组合恒等式  杨辉三角

         ↓
古典概型 ──→ 条件概率 ──→ 独立事件 ──→ 随机变量
  │            │            │            │
等可能       贝叶斯公式    互斥/独立    二项/正态分布
枚举法                      频率≈概率   期望/方差
```

#### 领域七：数列与导数初步（高中进阶）

```
等差数列 ──→ 等比数列 ──→ 数列求和 ──→ 极限思想 ──→ 导数
  │            │            │            │
通项公式     通项公式     裂项/错位相减   无穷逼近   f'(x)
前n项和      前n项和      数学归纳法              切线/单调性
                                                极值/最值
```

### 2.3 知识图谱数据结构

每个知识点是一个节点，边表示前置/后置关系：

```
知识点节点 = {
  id: "MATH-01-001",        // 编码：领域-章节-序号
  title: "虚数单位 i",
  domain: "数",              // 所属领域
  level: "高中",             // 初中/高中
  difficulty: 3,             // 难度 1-5
  prerequisites: ["平方根", "负数概念", "实数"],  // 前置知识
  next: ["复数运算", "欧拉公式"],                // 后继知识
  visualType: "canvas|svg|3d|static",           // 可视化类型
  summary: "i²=-1，乘以 i 相当于在复平面逆时针旋转 90°",
  milestones: [              // 关键节点/总结点
    { type: "section_end", title: "复数章节小结" },
    { type: "domain_end", title: "数的领域总结" }
  ]
}
```

### 2.4 总结节点（里程碑/Milestone）

每个章节末尾、领域末尾、阶段末尾设 **总结节点**：

| 层级 | 总结类型 | 示例 |
|------|---------|------|
| 章节末 | 章节小结 | "数轴与相反数 小节总结" |
| 大章末 | 单元总结 | "数与式 单元总结" |
| 领域末 | 领域回顾 | "【数的世界】全景回顾" |
| 初中/高中末 | 阶段总复习 | "初中数学知识全景图" |

总结页面形式：
- **知识地图**：该章节所有知识点的关系图（D3.js 力导向图）
- **一图胜千言**：核心结论用一张图总结
- **易错点整理**
- **经典例题回顾**

#### ④ 训练与考试系统

```
训练体系
  ├─ 知识点随堂练（3-5题/知识点，即时反馈）
  ├─ 章节巩固练（10-15题，覆盖本章所有知识点）
  ├─ 单元综合练（20题，跨章节）
  └─ 易错题专练（基于错题本推荐）

考试体系
  ├─ 单元测试（30分钟/20题）
  ├─ 期中/末模拟（90分钟/全卷）
  ├─ 中考模拟 / 高考模拟
  └─ 智能组卷（选范围/难度/题量，AI 排题）

题目类型：选择题 | 填空题 | 判断题 | 解答题（输入公式/手写识别）
答题模式：练习模式（有提示） | 考试模式（严格限时 + 防作弊）
```

#### ⑤ AI 辅助功能

- **AI 知识问答：** 选中知识点，问"为什么 i²=-1？" → AI 用通俗语言+图示解释
- **AI 错题解析：** 做题错了，AI 分析错误原因（概念不清/计算失误/审题问题）
- **AI 学习路径推荐：** 根据错题和弱项，推荐优先复习的知识点
- **AI 智能组卷：** 输入"初中函数综合，中难度，15题" → 自动生卷

## 九、潜在挑战与应对

| 挑战 | 应对方案 |
|------|---------|
| 知识内容量大，人工录入慢 | 先用 AI 生成初稿，人工审核 + 图表补充 |
| Canvas 互动演示开发成本高 | 建立可复用的互动演示组件库，P0 先做最重要的 |
| 数学公式渲染 | MathJax / KaTeX 客户端渲染，公式存 LaTeX 格式 |
| 3D 立体几何交互性能 | Three.js + 模型优化，移动端降级为 2D 演示 |
| 考试防作弊 | Web 端切屏检测 + App 端锁定模式 |
| 内容版权 | 所有知识点为自创/改编，图文素材自绘或用开源素材 |

## 十、总结

**KnowledgePower** 不是又一个刷题 App，而是一个 **可以"看见"数学** 的学习平台。

- 📖 **看得到**：每个知识点都有图和交互演示
- 🔗 **连得上**：知识点从哪来到哪去，一目了然
- 🧪 **练得会**：从随堂练到综合考，层层递进
- 🤖 **问得到**：AI 随时解答，"为什么"不再无解
- 📱 **带得走**：Web 深度学 + App 碎片时间练
